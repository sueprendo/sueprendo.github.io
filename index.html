Coding and video editing

This webpage was coded using HTML5 and CSS3 for practice purposes. I'm working on eventually creating and deploying a website where I can play and practice using some of the latest tools and tech.

Why though?

Learning to code with front end programming languages is one of those multi-beneficial endeavours that will help me to learn more about computer technology while allowing me to make and create tangible work simultaneously. Not only does it satisfy my own itch to learn and create, it will also bring many benefits to our eLearning development processes such as:

  • better problem solving and troubleshooting skills
  • a better understanding of variables and triggers
  • the potential for more customized eLearning developed in popular development tools
  • the ability to understand and use open-source development tools
  • the ability to program static websites and eventually more complicated websites and apps
  • honed crossover skills and process knowledge
  • making use of programming resources, frameworks and online communities to code and develop more efficiently
